Privacy Policy for GitPulse

Effective Date: April 25, 2025

Overview

GitPulse is a Chrome extension designed to help developers monitor activity and changes across Git repositories. It provides insights like commit trends, repository health, and team contributions, all while prioritizing user privacy.

Data Collection and Usage

  • Local Storage: GitPulse stores configuration and preferences locally in your browser.
  • Permissions: Only requests permissions necessary to display repository information and updates.
  • No Personal Data: GitPulse does not collect personal information or credentials. Repository data is accessed via public APIs with your explicit authorization.

Data Sharing

GitPulse does not share, sell, or disclose any user-specific data to third parties. All repository data is retrieved securely through the official APIs.

Security

All settings and usage data are stored locally in your browser. GitPulse does not transmit sensitive information externally, ensuring complete privacy.

Changes to This Policy

We may update this Privacy Policy periodically. Any changes will be posted here with an updated effective date.
